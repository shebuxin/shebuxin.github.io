# Buxin She Personal Website

This repository contains the source for [https://shebuxin.github.io](https://shebuxin.github.io), a personal academic website built with Jekyll and GitHub Pages.

## Local development

1. Install Ruby, Bundler, and Node.js.
1. Run `bundle install`.
1. Run `bundle exec jekyll serve`.

The site will be available at `http://localhost:4000`.
